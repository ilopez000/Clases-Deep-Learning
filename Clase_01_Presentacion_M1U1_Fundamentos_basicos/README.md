## Cómo usar los notebooks

1. Sube el archivo `.ipynb` a [Google Colab](https://colab.research.google.com) (Archivo → Subir cuaderno) o ábrelo desde GitHub.
2. Ejecuta las celdas en orden con **Shift + Enter**.
3. Lee cada celda de texto antes de ejecutar la celda de código que tiene debajo.

Los notebooks incluyen la teoría necesaria junto con los ejemplos, para que puedan seguirse de forma autónoma.
